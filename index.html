대입 컨설팅 엔진

자료 입력부터 리포트까지,
7단계가 하나의 흐름으로

생기부 PDF·사진을 올리면 AI가 성적·세특·창체를 자동 판독합니다. 이후 모든 단계가 서로의 결과를 이어받습니다.

1

생기부 자동 판독 — 흐린 사진도, 애매하면 비워둡니다

학기별 성적표를 행 단위로 정독하고, 판독이 불확실한 값은 추측 대신 표시합니다. 9등급제·5등급제(2022 개정)를 학년에 맞게 자동 구분합니다.

PDF·사진 다중 업로드원본 미저장 · 파싱 결과만불확실 값 표시
2

진단이 곧 전략 — 학과 매칭이 역설계에 자동 반영

5역량 진단과 진로·학과 매칭 결과가 합격 역설계(지원가능 대학 6선)의 근거로 주입됩니다. 성별(여대 자동 제외)·전국 단위 매칭 규칙을 지킵니다.

reach/fit/safe 6선대입은 지역 제한 없음지역인재전형 참고 표시
3

2022 개정 교육과정 네이티브 — 이수과목 검증·선택과목 로드맵

계열 권장과목 DB 대비 이수 현황을 점검하고, 위계를 지키는 남은 학기 로드맵을 설계합니다.

고교학점제 대응위계 결손 경고미개설 대체안 제시
4

학생부 디자인 — 성취기준 안에서, 대필 없이

과목별 탐구를 교육과정 성취기준 범위 안에서, 진로로 수렴하는 하나의 성장 서사로 설계합니다. 학생이 실제 수행하고, 타이핑 무결성으로 진정성을 증명합니다.

세특 문장 생성 안 함수행평가 루브릭 예상 채점타이핑 DNA 무결성
5

면접 — 생기부에서 나올 질문을, 꼬리질문까지

서류기반·전공기초·인성 3유형 예상 질문과 출제 근거·답변 구조 가이드·꼬리질문을 생성하고, 모의답변에 면접관 관점 피드백을 제공합니다.

문항별 출제 근거 태그답변 대필 없음 · 구조만제출↔코멘트 반복 루프
CONSULTANT ↔ STUDENT

속도가 아니라, 협업 구조입니다

아르케의 본질은 "AI가 대신 써주는 것"이 아니라 컨설턴트와 학생이 함께 완성하는 루프입니다. 전달·제출·코멘트가 오갈 때마다 🔔 알림이 도착합니다.

🧑‍🏫

컨설턴트가 설계

탐구 주제·면접 문항·자소서 설계도를 검토하고 학생에게 전달

🧑‍🎓

학생이 수행

직접 작성·제출 — 타이핑 무결성으로 대필·붙여넣기 검증

🤖

AI가 코칭

방향·논리·오류를 짚는 소크라테스식 코칭 (대신 쓰지 않음)

🔁

컨설턴트 피드백과 함께 라운드 반복 — 성장 이력이 쌓임

고입 컨설팅 엔진 특목·자사고

중등부는 고입 엔진이 따로 돕습니다

같은 화면에서 학년만 입력하면 중학생은 고입 트랙으로 자동 분류됩니다.

🎯

지원가능 리스트업 — 내신 환산 기반

중학교 내신을 환산해 특목고·자사고·일반고 지원 가능선을 잡고, 지역 규정을 반영해 리스트업합니다. (고입은 지역 규칙 적용)

🧩

학생부 성향 매칭

학교별 인재상·선발 스타일과 학생 활동 성향의 적합도를 분석합니다.

📝

자소서 설계 → 작성 → 평가 루프

설계도를 컨설턴트가 검토·전달하면 학생이 작성하고, 4축 평가와 코멘트로 라운드를 반복합니다.

🎤

면접 준비

학교 유형별 예상 질문과 중학생 눈높이의 답변 가이드, 모의 답변 피드백을 제공합니다.
